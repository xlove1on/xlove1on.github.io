Markdown标题

两种形式：一种前面加#号
另一种形式：在下一行添加短线------------

Markdown 段落

换行直接回车或者空格两次

字体

在字体的两边添加*号

一个*号是斜体文字

两个**号是粗体文字

三个***号是斜粗体文字

分割线

三个以上的星号/减号/底线

删除线

在文字两边添加连个波浪线~~
如

~~Daxun日记~~

Daxun日记

下划线

在文字两边添加

如

<u>Daxun日记</u>

Daxun日记

链接

[Daxun日记](http://blog.daxun.art)

链接格式Daxun日记

表格

如：

|  表头   | 表头  |
|  ----  | ----  |
| 单元格  | 单元格 |
| 单元格  | 单元格 |
表头 表头
